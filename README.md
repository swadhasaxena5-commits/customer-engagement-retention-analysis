# Customer Engagement & Product Utilization Analytics Dashboard

## Project Objective

This project analyzes customer engagement, product utilization, and churn behavior using a European banking dataset. The goal is to identify retention drivers and provide actionable recommendations for customer loyalty improvement.

## Features

* Customer Activity vs Churn Analysis
* Product Utilization Analysis
* Geography-wise Churn Analysis
* Salary vs Balance Analysis
* High-Value Disengaged Customer Detection
* Credit Card Stickiness Score
* Relationship Strength Index
* Retention Segmentation
* Interactive Streamlit Dashboard

## Technologies Used

* Python
* Pandas
* Streamlit
* Plotly
* Matplotlib
